# frame_airdrop_script
frame空投查询教学

官网：https://www.frame.xyz/governance

从官网上查空投，需要5步骤：

1、链接钱包   --》      2、钱包签名（重点）   --》    3、连接推特    --》     4、发推      --》      5、查询空投

这些都是障眼法

当你   链接钱包、用钱包签名后，已经可以在浏览器的网络请求中，发现一条返回具体空投数量的请求：“authenticate”  里面包含了你的{交易金额、支付的佣金、排名、空投数量}
